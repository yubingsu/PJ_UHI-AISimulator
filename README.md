# PJ_UHI-AISimulator
[2025.09 ~ 2025.12] '서울시 행정동별 열섬 현상 완화 지원 AI 시뮬레이터 구축'프로젝트

## 프로젝트 배경
- 서울시 열섬 현상 심화 (50년간 평균기온 2.4°C 상승, 도심-외곽 온도 격차 7°C)
- 서울시 행정동 단위의 열섬 발생 원인 및 정도를 파악한 후 열섬 대응 정책 지원 진단 구축 필요
- 정책 의사결정자들의 열섬 대응 우선순위 수립 및 완화 정도 파악 어려움

## 주요 성과
1. **데이터 수집 및 전처리**: 4개 경로에서 열섬 영향 요인 변수를 수집·정합성 검증·결측 개선하여 **최종 25개의 신뢰도 높은 변수** 구성
2. **변수 분류 체계 구축**: 25개 변수를 지표·기상·도시구조·인프라·사회·경제·시간 5가지로 분류한 후, 관련 정책 의사결정자 연결
3. **ML 모델링**: 3종 모델(RandomForest, XGBoost, LightGBM) 성능 비교 후 LST(지표면온도) 기반 고품질 데이터 구축으로 중속변수 정합한 예측값·설명력 확보
4. **정책 도구 제공** : SHAP 분석으로 LST 변화에 영향을 미치는 변수 순위 및 영향 방향과 정도 파악, 정책 의사결정자에게 맞춤형 열섬 대응 전략 도출

## 핵심 인사이트
- 최종 모델(LightGBM) test 정확도: **0.92%** (매우 높은 설명력)
- 행정동·월 단위 서울시 LST 변화의 영향 요인을 정량적으로 해석하여 정책 의사결정자의 맞춤형 열섬 대응 전략 도출

## 실행 환경
* Python 3.x
* Pandas, Scikit-learn, XGBoost, LightGBM, SHAP

## 데이터 출처
* [AI Hub] 기후변화 대응 열지도 통합 데이터
* [공공데이터포털] 서울시 행정동 경계, 건물, 인구 등
* [Landsat 8/9] 위성 영상 데이터 (2020~2024, 426개 이미지)
